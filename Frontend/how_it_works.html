How Credible Works

1. User Input

The user pastes a news article or claim into the input box on the Credible platform.

2. Request Sent to Backend

The text is sent to the Flask backend through a JavaScript Fetch API request.

3. AI Model Analysis

The backend forwards the news text to the Gemini AI model for credibility analysis.

4. Prediction

The AI model analyzes the claim and generates a verdict with confidence score.

5. Result Display

The result is returned to the frontend and displayed to the user.
